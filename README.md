Вас приветствует проект WeeRGet.

Этот сайт создан для получения необходимых навыков людям которые собираются идти на собеседование.
Тут вы можете посмотреть различные страницы с документацией о языке программировании Python, собеседовании, оставить идею для сайта.
Так же этот проект постоянно дорабатывается, улучшается и становится лучше.


Для запуска:

1) Установите виртуальное окружение:

Windows: python -m venv venv

Linuxc:

MacOs:

2) Активируйте его:

Windows: source venv/Scripts/activate

3) Установите библиотеки из requirements.txt:

Windows: pip install -r requirements.txt

4) Перейдите в weerGet/weerGet

Выполните: python manage.py runserver

5) Оставьте своё мнение о сайте в форме и предложите свои идеи по улучшению!
